如何搭建ngrok穿透内网

May 4, 2016 | Hits

ngrok 是一个反向代理，通过在公共的端点和本地运行的 Web 服务器之间建立一个安全的通道。ngrok 可捕获和分析所有通道上的流量，便于后期分析和重放。简单来说就是可以让你的本地服务暴露在外网上面，可以通过外网访问，这是我们经常需要用到的功能。

为何

因为宽带运营商只是提供动态的公网IP,我们使用路由器DHCP(Dynamic Host Configuration Protocol，动态主机配置协议)来获得诸如192.168.0.2的IP地址进行上网。假如我们想远程登录链接在家中路由器下的树莓派是不能访问的，这时候我们就要考虑ngrok或花生壳来达到远程访问的效果。
